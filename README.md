# create-preact

Create a Vite-powered Preact app in seconds

<p align="center">
  <img src="https://github.com/preactjs/create-preact/blob/master/media/demo.gif?raw=true">

## Usage

```sh
$ npm init preact

$ yarn create preact

$ pnpm create preact

$ bun create preact
```
